# Inert

Minimalistic tower defense in the browser. [Try it!](https://inert.thomasset.me)

## Dev

```shell
git clone git@github.com:CorentinTh/inert.git
cd inert
npm i
npm run dev
# open http://localhost:1234/index.html
```
